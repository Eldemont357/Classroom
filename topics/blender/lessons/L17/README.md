#  Preliminary Presentation (Blender)

## Описание урока
На уроке проходит **предварительная презентация проекта** в Blender.  
Основной фокус — на процессе работы и понимании инструментов.

## Задача ученика
Подготовить презентацию проекта и рассказать:
- идею проекта,
- этапы работы,
- какие инструменты использовались.

## Формат презентации
Презентация должна включать:
1. Титульный слайд (имя, проект)
2. Цель проекта
3. Идея и референсы
4. Процесс работы (скриншоты)
5. Использованные инструменты
6. Выводы

## Требования
- Минимум текста, больше визуального материала
- Обязательны скриншоты из Blender
- Презентация должна быть аккуратной и структурированной

## Результат урока
После урока ученик:
- умеет объяснять свой процесс работы,
- понимает, какие навыки освоил,
